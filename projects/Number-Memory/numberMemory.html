Number Memory guide



You will start off the game by selecting a difficulty. The harder the difficulty, the lower time you have to memorise the number.

A number will then flash up on the screen and you have to try and memorise the number before the timer runs out. You will then be given a screen which will allow you to type in the number that you memorised. If you get the answer correct, you will be loaded into the next level and will be given a larger number to remember. If you get the number incorrect, you will be subtracted 1 life and will be given the opportunity to retry the level.

Good Luck!
